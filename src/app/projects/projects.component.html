Incluvie: Actor Data Collection

Incluvie is a platform that promotes and celebrates diversity within Hollywood. Our project with Incluvie is focused on actors and actresses. The task is to create a method that extracts specific data parameters related to gender and ethnic backgrounds of the actor. We created code and data visualizations to display diversity in Hollywood. Incluvie uses the datasets/code to compile actor profiles for its users. Utilizing this data, users can easily determine the kind of gender and cultural diversity stance a movie holds by viewing a movie’s cast and the actors’ gender/ethnic information.

Technologies: Python, APIs (TMDB, Wikipedia)

Digaai: Name-Ethnicity Classification

Digaai is a digital platform that documents the Brazilian diaspora through media (eg. text, audio, video and images). The purpose of our project was to assist Digaai in locating and classifying locations in the US made of the Brazilian immigrant population. We do so by identifying Brazilian names from a list of names and differentiating between Hispanic/Latino names that typify as Brazilian and those that do not.

CS 411 Project

Trippppr is a travel itinerary app. It helps you plan out your trip by searching events, restaurants, points of interests and flights within one platform. You simply enter trip details into the search box and the dates you will be there. A list of events happening in that area at the same time via will be generated via the Eventbrite API. Google Maps will return points of interest in the destination area. Zomato will return the popular restaurants in the area. IATA Geo is used to help search flights for the given set of dates. We also use Facebook OAuth to store the user's data. Each user will have a profile in which they can access all previously saved trip itinerarys.

Technologies: APIs (Eventbrite, Zomato, Google Maps Places, IATA Geo), MERN (Mongo, Express React, Node) stack

Quadratic Solver iOS App

Quadratic solver is a simple iOS app that was built with the mission to assist students with their math homework. The student just enters all three of the variables in the corresponding input boxes and press solve. The x value will be printed on the bottom. The app handles error inputs and displays error statements if the inputs are invalid or if there is not solution. I built this when I first started learning Swift as a small app to test my beginner Swift knowledge.

Technologies: Swift, Xcode
